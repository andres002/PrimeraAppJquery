Draggable (Cursors and Position cursors)

Ejemplo con divs, cursors & containment parent

El cursor siempre estará relativamente al centro del div.

El cursor es crosshair y se encontrará a -5px, -5px del div.

El cursor estará en la posición "x" del cursor y "y" bottom del div.

$( "selector" ).draggable({ cursor: "tipo de cursor", cursorAt: { top: position, left: position... } });
$( "selector" ).draggable({ cursor: "tipo de cursor", cursorAt: { top: -5, left: -5 } });
$( "selector" ).draggable({ cursorAt: { bottom: 0 } });

Draggable (Solo en posiciones específicas)

Ejemplo con divs, cursors & containment parent

Puedes moverme :D

Puedes moverme unicamente con la barra de la cabecera del div.

$( "#div u objeto" ).draggable({ handle: "#div u objeto", containment: "parent" });
$( "div u objeto" ).disableSelection(); - Se deshabilita la selección
