jQuery hide() and show()


Los efectos hide y show nos sirven para ocultar o mostrar elementos HTML




        Como implementarlos:

            $("#hide").click(function(){
                $("p").hide();
            });

            $("#show").click(function(){
                $("p").show();
            });

Otras maneras


        Como implementarlos:

            $("#hide").click(function(){
                $("p").hide(1000); "En ms"
            });

            $("#show").click(function(){
                $("p").show("slow");
            });

Efectos Fading (Aparecer y Desaparecer)


Ejemplos con fadeIn(), fadeOut(), fadeToggle(), fadeTo()

                    

        Como implementarlos:

            $("button").click(function(){
                $("#div1").fadeIn();
                $("#div2").fadeOut("slow");
                $("#div3").fadeToggle(3000);
                $("#div4").fadeTo("slow", 0.7);
            });

jQuery Animations (Animaciones)

Ejemplos con animate()

                    

        Como implementarlos:

            $("button").click(function(){
                $("div").animate({
                    left: '250px',
                    height: '150px',
                    width: '150px'
                });
            });

Otras maneras


        Como implementarlos:

            $(document).ready(function(){
                $("button").click(function(){
                    $("div").animate({
                        height: 'toggle'
                    });
                });
            });

jQuery Callback Functions (Llamar de vuelta)

Ejemplos con llamadas a funciones.

                    
        Como implementarlos:
            $("button").click(function(){
                $("p").hide("slow", function(){
                    alert("The paragraph is now hidden");
                });
            });

Otras maneras


        Como implementarlos:


            $("p").show("slow", function(){ "algo" });
            $("p").toggle("slow", function(){ "algo" });
            $("p").toggle(1000, function(){ "algo" });

jQuery Method Chaining (Encadenamiento)

Ejemplos con declaraciones multiples en el mismo elemento.

                    

        Como implementarlos:

            $("button").click(function(){
                $("#p1").css("color", "red")
                            .slideUp(2000)
                            .slideDown(2000);
            });
